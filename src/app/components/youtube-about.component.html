About

This is a simple youtube web player and desktop player based on angular-cli 1.7.3, ngx-youtube-player 6.0.0 (by orizens, used for youtube iFrame API), SCSS (CSS3), HTML5.

This player is compatible only with Chrome/webkit browsers.

The project status is beta.

Plugins used: ngu-carousel (https://github.com/sheikalthaf/ngu-carousel) ng2-dragula
